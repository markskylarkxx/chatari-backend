This requires a Meta Developer account and goes through a few steps — follow exactly.

Step 1 — Create a Meta Developer Account
Go to developers.facebook.com → Log in with your Facebook account → Click "Get Started" → Verify your account.

Step 2 — Create a Meta App

Go to developers.facebook.com/apps
Click "Create App"
Select "Business" as the app type
Give it a name — e.g. "Chatari"
Click "Create App"


Step 3 — Add WhatsApp to your App

Inside your new app dashboard, scroll down to find "WhatsApp"
Click "Set up"
It will ask you to connect a Meta Business Account — create one free if you don't have one
Click through until you reach the WhatsApp API Setup page


Step 4 — Get your credentials
On the WhatsApp API Setup page you will see:
Phone Number ID:     1234567890123
WhatsApp Business Account ID (WABA ID): 9876543210
Temporary access token: EAAxxxxxxxxxxxxx
Copy all three and add to your .env.local:
envMETA_PHONE_NUMBER_ID=1234567890123
META_WABA_ID=9876543210
META_PERMANENT_ACCESS_TOKEN=EAAxxxxxxxxxxxxx

Step 5 — Add a real phone number
On the same page, click "Add phone number" and add your Nigerian number. Meta will send a verification code via SMS or call. Enter it to verify.

Step 6 — Register your webhook

In your Meta app, go to WhatsApp → Configuration
Under Webhook, click "Edit"
You need a public URL — since you're on localhost, run this first:

bashnpx ngrok http 3000
It gives you a URL like https://abc123.ngrok.io — copy it.

Set Webhook URL to:

https://abc123.ngrok.io/api/webhook

Set Verify Token to:

whatflow_webhook_secret

Click "Verify and Save"
Under Webhook Fields, click "Manage" and enable "messages"


Step 7 — Update your dev business record
Run this in Supabase SQL Editor — replace the phone number ID with your real one:
sqlUPDATE "Business"
SET 
  "whatsappPhoneId" = 'YOUR_PHONE_NUMBER_ID',
  "whatsappPhoneNumber" = 'YOUR_NIGERIAN_NUMBER'
WHERE "id" = 'dev-business-id';
